# novela_2.0

Браузерний редактор для створення інтерактивних візуальних новел із базовим робочим простором.

## Швидкий старт
1. Відкрийте `index.html` у сучасному браузері (Chrome, Edge, Firefox). Інтернет не потрібен.
2. Додайте або відредагуйте сцени у лівій панелі, перемістіть їх на полотні та налаштуйте переходи у правій панелі.
3. Використовуйте «Попередній перегляд», щоб побачити логіку, та кнопки експорту для збереження результату.

## Основні можливості робочої області

### Головна панель навігації (верх екрана)
- **Профіль** — зарезервовано для налаштувань та виходу з облікового запису.
- **Допомога** — відкриває довідку з підказками.
- **Попередній перегляд** — швидкий перегляд першої сцени та наявних переходів.
- **Експорт** — вивантаження проєкту у HTML та JSON.

### Ліва бокова панель (панель інструментів)
- **Сцени** — перелік усіх сцен у проєкті, кожна у вигляді картки з назвою та коротким описом.
- **+ Додати сцену** — модальне вікно для створення нової сцени (назва й текст обов’язкові, фон опціональний).
- **Зміни** — список змінних для умовної логіки та переходів.
- **+ Додати зміну** — додає нову змінну для умовних операцій.
- **Швидке збереження** — автоматичне збереження в браузері та можливість скопіювати посилання на проєкт.
- **Скопіювати посилання на проєкт** — створює посилання зі станом (scene/variable) для подальшої роботи.

### Центральна область (робоче полотно)
- Відображення сцен у вигляді карток із заголовком, коротким текстом та переходами.
- **Drag-and-Drop**: переставляйте сцени для зміни порядку.
- Підсумок переходів: кожен вибір показує, до якої сцени він веде.

### Права панель (редагування сцени)
- **Редагувати сцену** — активується після вибору сцени; доступні поля для тексту, фону та медіа.
- **Додати вибір** — створення варіантів, що ведуть до інших сцен; для кожного вибору можна вказати цільову сцену.
- **Шаблони сцени** — готові варіанти («вступ», «конфлікт», «закінчення») для швидкого наповнення текстом.
- Панель зміни фону сцени — поле для URL або короткого опису фону.

## Алгоритм роботи
1. **Створення нового проєкту**: відкрийте робочу область — з’явиться стартова сцена «Вступ».
2. **Додавання нової сцени**: натисніть «+ Додати сцену», заповніть назву й текст, за бажанням фон; збережіть.
3. **Редагування сцени**: оберіть сцену в списку або на полотні, відредагуйте текст, фон, медіа та вибори в правій панелі.
4. **Зв’язування сцен**: для кожного вибору виберіть цільову сцену; зв’язки відображаються на полотні в картках.
5. **Тестування**: натисніть «Попередній перегляд», щоб перевірити переходи.
6. **Експорт**: скористайтеся кнопками експорту (HTML або JSON) на верхній панелі після завершення історії.

## Подальші кроки
- Додати розширені типи медіа (галереї, аудіоплеєр), стан користувача та публікацію.
- Візуалізувати зв’язки між сценами лініями на полотні та додати фільтри за статусом сцени.
